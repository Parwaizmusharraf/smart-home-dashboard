
# 🏠 Smart Home Dashboard – UI/UX Project

A modern, responsive smart home control dashboard designed during my UI/UX internship.

## 🚀 Features
- Light ON/OFF toggle
- Fan speed control (slider)
- Temperature display
- Door lock control
- Sidebar navigation
- Responsive layout (desktop + mobile)

## 🛠️ Tools Used
- Figma
- Adobe XD (optional)
- Google Fonts
- ColorHunt palette
